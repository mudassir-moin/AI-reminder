# AI-reminder
AI-assisted reminder application focused on intelligent task scheduling, priority-based notifications, and adaptive reminders based on user behavior.

## Technologies & Tools
React (Frontend), Node.js & Express (Backend), REST APIs, Lightweight Database (Prototype), AI-assisted development via Bolt, Cloud-based prototype deployment.

## What I Am Learning
- Application structure and workflows
- Frontend vs backend responsibilities
- Using APIs in applications
- AI-assisted development methodologies

## Current Status
This project is currently under development. A functional prototype has been deployed using Bolt.
